# PM BANK — Dark Luxury Landing Page
High-end black & gold landing page built for a QR-first payments brand.

## Files included
- index.html
- styles.css
- assets/pm_logo.jpg (generated placeholder — replace with your real logo if you prefer)
- README.md

## How to use
1. Unzip and upload all files to the root of your GitHub repository (or Hosting provider).
2. Make sure `index.html` is in the root (not in a subfolder).
3. Enable GitHub Pages in repository Settings → Pages → branch `main` → folder `/root`.
4. Replace `assets/pm_logo.jpg` with your real logo file if you have one (same filename).
